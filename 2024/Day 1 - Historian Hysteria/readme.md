# Day 1 - Historian Hysteria

**Language:** C

**[Link to Problem Page](https://adventofcode.com/2024/day/1)**

**Brief:** Given two arrays of integers of length _N_, sort the lists and compute the absolute differences of the elements in corresponding indices of the sorted arrays. Sum these absolute differences for the final integer solution.

---

**Sample Input:**
| L1 | L2 |
| -- | -- |
| 3 | 4 |
| 4 | 3 |
| 2 | 5 |
| 1 | 3 |
| 3 | 9 |
| 3 | 3 |

**Sample Output:**<br>
|1 - 3| = 2<br>
|2 - 3| = 1<br>
|3 - 3| = 0<br>
|3 - 4| = 1<br>
|3 - 5| = 2<br>
|4 - 9| = 5<br>

2 + 1 + 0 + 1 + 2 + 5 = **11**
